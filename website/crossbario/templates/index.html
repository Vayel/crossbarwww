Crossbar.io
Open-source polyglot application router.

Crossbar.io is a polyglot, open-source application router which implements the Web Application Messaging Protocol (WAMP).

WAMP implements asynchronous Remote Procedure Calls and Publish & Subscribe, with WebSocket as a preferred transport.

Crossbar.io enables the easy implementation of distributed, multi-client and server applications such as multi-user database-driven business applications, sensor networks (IoT), instant messaging or MMOGs (massively multi-player online games)
