Roadmap

Crossbar.io is completely functional and usable as a WAMP router with advanced functionality.

At the same time, Crossbar.io is a work in progress. We have a lot of additional functionality planned.

Below is an high-level overview of where we want to go with Crossbar.io.

Database Connectors

Modern databases can be much more than simple data stores. They offer fully featured programming languages which are optimized for the processing of stored data.

With a database connector, we want to offer as much of the WAMP functionality as possible for code which runs directly in your database.

This means that you can move your business logic into the database, close to your data.
Using the Web server integrated into Crossbar.io, you will be able to write database applications without the need for an additional application server.

We are planning on offering database connectors for

  • PostgreSQL
  • Oracle

With PostgreSQL, you get pluggable programming languages. For example, you can use plv8js to run JavaScript in the database.

REST bridging

While WAMP is all you need to develop your own applications, Web services today use REST APIs.

With REST bridging, Crossbar.io will be able to translate between WAMP and REST.

You could, for example, post a pubsub event from traditional Web application and have this delivered to WAMP subscribers, or call a REST API endpoint based on a WAMP remote procedure call.
